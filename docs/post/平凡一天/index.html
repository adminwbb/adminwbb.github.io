平凡的一天

October 27, 2020 adminwbb

1 minute read

  • Twitter

  • Facebook

  • Email

  今天又是个劳累的一天，我的组长又不来上班，把我一个人丢在实验室里。我有点心烦，但是那有什么办法呢？干就完了。   上午安排上样，车间主任暴跳如雷的向我冲来，仿佛要一拳把我打到在地，他的双眼中我看到了那种对数据的渴望。他恨不得，把各种数据收入眼中。

  我的组长，我估计现在正在宿舍睡大觉呢！尽然让我这个这个未成年的孩子，干六天活，天理何在。我发誓不是不报只是时候未到。

  下午干活，又是几十个样，明天又要上班了，想到我的组长又能睡大觉我就恶向胆边伸。
