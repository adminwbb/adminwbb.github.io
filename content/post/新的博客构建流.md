## 最近更新 (2026-05-23)

### 1. GitHub Actions CI/CD 迁移
- 部署从手动脚本迁移到 GitHub Actions 自动部署
- 新增 `deploy.yml` workflow，master 分支 push 后自动构建部署到 GitHub Pages
- 使用 Hugo extended 版本支持 SCSS 编译

### 2. Markdown 自动格式化
- 新增 `format-markdown.yml` workflow
- 每次 `.md` 文件变更自动：
  1. 按文章标题重命名文件
  2. 用 Prettier 格式化 Markdown
- 防止 bot 提交触发无限循环：`github.actor != 'github-actions[bot]'`

### 3. 文章文件重命名

- 解决 Hugo 构建警告（languageCode、language 配置弃用）

- 删除 `newpost.sh` 和 `rename.py`（已迁移到新脚本）

### 6. CLAUDE.md

## 技术细节

**deploy.yml:**
```yaml
- name: Setup Hugo
  uses: peaceiris/actions-hugo@v3
  with:
    hugo-version: 'latest'
    extended: true

- name: Build
  run: hugo --minify
```

**format-markdown.yml:**
```yaml
jobs:
  format:
    runs-on: ubuntu-latest
    if: github.actor != 'github-actions[bot]'  # 防止无限循环
    steps:
      - name: Rename files by title
        run: python3 scripts/rename_by_title.py content/post

      - name: Format Markdown
        run: prettier --write "**/*.md"
```

**scripts/rename_by_title.py:**
- 读取 Markdown 文件 frontmatter 中的 `title` 字段
- 清理文件名（移除非法字符）
- 处理重名（自动加序号）
